This is a list of all scripts and their functions
0-iam_betty - switch current user
1-who_am_i - prints username of current user
2-groups - print all current user groups
3-new_owner - changes file owner
4-empty - creates a blank file
5-execute - add execute permission to a file
6-multiple_permissions - add different permissions to multiple users
7-everybody - adds execute permission for all users
8-James_Bond - sets 007 permissions to hello
9-John_Doe - sets mode to 753 for hello
10-mirror_permissions - mirror another file permission to your file
11-directories_permissions - adds execute permission to all subdirectories
12-directory_permissions - creates a directory with custom permissions
13-change_group - changes group owner for file 'hello' to school
